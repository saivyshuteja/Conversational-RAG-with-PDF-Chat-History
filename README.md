# Conversational-RAG-with-PDF-Chat-History

•	Built an interactive Conversational RAG (Retrieval-Augmented Generation) application using Streamlit, enabling users to upload PDFs and query their content in real time.
•	Implemented document ingestion, text-splitting, vector embeddings, and semantic search using HuggingFace embeddings and Chroma vector store.
•	Integrated Groq’s LLM (Llama 3.1) with a history-aware retriever, improving contextual understanding across multi-turn conversations.
•	Developed a persistent chat history system using RunnableWithMessageHistory, enhancing continuity and user experience.
•	Designed a secure workflow using environment variables, PDF parsing, and fully dynamic, multi-file retrieval chains through LangChain Classic.
